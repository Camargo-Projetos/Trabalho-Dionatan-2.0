# 📌 Planejamento de Sprint - PetLife

## 1. Informações Gerais
- **Período:** 15 dias (sugerido)
- **Objetivo:** Estabelecer a base estrutural, fluxo de agendamento e operações comerciais básicas.

---

## 2. 🧠 Squad Camargo (Alta Prioridade)
**Foco:** Infraestrutura, Regras de Negócio e Segurança

- [ ] Definição do escopo (Objetivos PetLife)
- [ ] Diagrama de Casos de Uso (Atendente/Admin)
- [ ] Modelagem do Banco de Dados (Tabelas e Relações)
- [ ] Requisitos Funcionais (Listagem técnica)
- [ ] Requisitos Não-Funcionais (Performance/Segurança)
- [ ] Interface de Cadastro de Clientes (Validações/Máscaras)
- [ ] Interface de Cadastro de Pets (Formulários/Histórico)
- [ ] Regras de Agendamento (Lógica de conflito)
- [ ] Fluxo de Login (Permissões)
- [ ] Recuperação de Senha (Fluxo de e-mail)

---

## 3. 💼 Squad Sazon (Média Prioridade)
**Foco:** Operações de Loja, PDV e Monitoramento

- [ ] Módulo de Vendas (PDV / Frente de Caixa)
- [ ] Controle de Estoque (Alertas de falta)
- [ ] Lista de Serviços (Catálogo de preços)
- [ ] Dashboard Principal (Visualização de agendamentos)
- [ ] Sistema de Alertas (Vacinas / Vermífugos)
- [ ] Histórico de Saúde do Pet (Prontuário)
- [ ] Busca e Filtros (Pesquisa rápida)
- [ ] Permissões de Acesso (Diferenciação de cargos)
- [ ] Comprovante de Agendamento (Geração de ticket)
- [ ] Integração de Pagamentos (PIX / Cartão)

---

## 4. 📊 Squad NegoLoko (Baixa Prioridade)
**Foco:** Business Intelligence, Retenção e Suporte

- [ ] Relatório de Faturamento (Gráficos)
- [ ] Relatório de Mais Vendidos (Análise de produtos)
- [ ] Sistema de Fidelidade (Pontuação / Brindes)
- [ ] Notificações via WhatsApp (Lembretes)
- [ ] Cadastro de Fornecedores (Gestão de compras)
- [ ] Política de Cancelamento (Prazos / Regras)
- [ ] Backup de Dados (Automação de segurança)
- [ ] Estudo de Responsividade (Mobile / Tablet)
- [ ] FAQ / Ajuda (Guia do usuário)
- [ ] Fechamento de Caixa (Conferência diária)
