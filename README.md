  Sistema PetLife
O PetLife é um projeto de planejamento para um sistema de gestão de Pet Shops. O objetivo é facilitar o controle de agendamentos, o cadastro de animais e a venda de produtos.

 O que o sistema oferece:
Agendamentos: Controle de horários para banho e tosa.

Cadastros: Registro completo de clientes e pets.

Estoque: Gestão de vendas de produtos e acessórios.
